# pc_sales_project

## Project Overview
This project focuses on analyzing a sales dataset using SQL.  
The goal is to explore business performance, identify trends, and extract meaningful insights from the data.

The analysis is structured into three levels:
 Basic Queries
 Intermediate Analysis
 Advanced Insights


## Purpose of the Project
This project demonstrates:
 SQL proficiency for data analysis
 Ability to transform raw data into insights
 Understanding of business metrics such as revenue, profit, and performance tracking


## Dataset Description
The dataset contains sales transaction data including:
 Sales details (Sale Price, Cost Price, Discount)
 Customer information (Credit Score, Location)
 Product details (PC Make, Model, RAM, Storage)
 Sales performance (Sales Person, Department)
 Operational data (Purchase Date, Ship Date, Payment Method, Channel)


## Tools & Skills Used
 SQL (MySQL / SSMS)
 Data Aggregation
 Grouping & Filtering
 Window Functions
 Business Analysis


### Basic Queries
These queries focus on understanding the structure and distribution of the data:
 Total number of sales records
 Sales distribution by continent, country, and shop
 Distinct values (Shop Name, PC Make)
 Average, minimum, and maximum sale price
 Sales breakdown by payment method, channel, and priority


### Intermediate Analysis
These queries focus on business metrics and performance:
 Total revenue and total profit
 Average discount and finance amounts
 Revenue per product (PC Make)
 Revenue per shop and salesperson
 Average credit score per payment method
 Cost of repairs by department


### Advanced Insights
These queries extract deeper business insights:
 Profit and profit margins per sale
 Top-performing continent by revenue
 Pricing analysis by RAM and storage
 Highest priced PC model
 Shipping efficiency (delivery time analysis)
 Department performance comparison
 Identifying underpriced sales
 Ranking sales people using window functions


## Key Insights
 Certain continents generate significantly higher revenue than others
 Salesperson performance varies widely across departments and stores
